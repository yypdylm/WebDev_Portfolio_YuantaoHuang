Tech Stack & Tools

This project applies Design-Thinking methodology, focusing on modular development based on repository structure (e.g., homepage, portfolio modules) to ensure code maintainability and user experience. Below are the core technologies and tools used:

HTML5

HTML is the core of web development, defining the content and structure of a webpage. It organizes text, images, and multimedia through tags and supports page navigation. Together with CSS and JavaScript, HTML builds the framework and functionality of modern webpages.

CSS3

CSS (Cascading Style Sheets) is essential in web development, controlling the visual presentation and layout of a webpage. It styles HTML elements with properties like colors, fonts, and spacing, ensuring an attractive and consistent design. By working with HTML and JavaScript, CSS enhances user experience through responsive and visually appealing interfaces.

JavaScript

JavaScript is a key programming language in web development, adding interactivity and dynamic functionality to webpages. It enables features like form validation, animations, and real-time updates by manipulating HTML and CSS. Integrated with HTML and CSS, JavaScript creates engaging, user-driven web experiences.
